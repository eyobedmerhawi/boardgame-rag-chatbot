# BoardGame RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot that answers board game rule questions using semantic search and grounded LLM responses.

## Features
- Semantic search using ChromaDB
- Sentence Transformer embeddings
- Groq LLM integration
- Gradio web interface
- Support for 9 board games
- Custom Connect 4 rulebook integration

## Tech Stack
- Python
- ChromaDB
- Sentence Transformers
- Groq API
- Gradio

## Example Queries
- What happens when you roll a 7 in Catan?
- How do you get out of Jail in Monopoly?
- How do you win in Connect 4?
